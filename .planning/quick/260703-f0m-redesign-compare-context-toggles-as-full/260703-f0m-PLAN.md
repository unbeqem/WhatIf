# Quick Task 260703-f0m — Option-row toggles

## Problem

The "Compare two paths — Pro" and "Add context" toggles on the decision form are
tiny all-caps `text-fg-mute` text links — the most muted token in the theme — so
users barely notice these features exist.

## Solution (user-selected: full-width option rows)

In `components/SimulateForm.tsx`, replace the two inline text-link toggles with a
single bordered container holding two full-width `OptionRow`s:
- Icon chip (lucide `Scale` for compare, `Sparkles` for context)
- Title (readable, sentence case) + one-line description
- Right-aligned badge ("Pro" / "optional") and a `Plus`/`Minus` indicator
- Active state (compareMode / showContext) tints the row + highlights icon/indicator

Preserve behavior: non-subscriber compare click still opens the upsell; context
row still expands the ChipRow panel; the upsell block and context panel render
below the container. No logic/state changes — purely presentation + a new
`OptionRow` helper.

## Verify

- `npx tsc --noEmit` clean, `npx vitest run` green
- Toggles clearly visible; active/inactive states legible; upsell + context still work
